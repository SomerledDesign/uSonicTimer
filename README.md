# uSonicTimer
An arduino code project to create a timer for my cheap ultrasonic cleaner.  It incorporates a Nokia 5110 to display countdown time.  Time setting is menu driven using [educa8s.tv](https://www.youtube.com/@Educ8s) [menu tutorial](https://youtu.be/ak5TsUFhyf8?si=9JEMm8WbRyF4dVVC).  A rotary encoder is used for navigating the menu.
